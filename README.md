# Spotify Data Storytelling

## Group Members

## Project Description
This project analyzes Spotify track data to explore relationships between audio features and song popularity.

## Dataset
Source:
https://www.kaggle.com/datasets/yashdev01/spotify-tracks-dataset

## Research Questions
1. Which genres have the most popular songs?
2. Does danceability affect popularity?
3. Are energy and loudness correlated?
4. What is the distribution of song popularity?
